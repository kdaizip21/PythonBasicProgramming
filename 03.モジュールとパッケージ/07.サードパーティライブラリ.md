## サードパーティライブラリ

- setup.pyで固めたパッケージ
- 一般では[pypi](https://pypi.org/)で配されている

## pip install

- pip コマンドでオフィシャルpypiからダウンロード、インストールする


```sh
$ pip install termcolor
```



## ライブラリの使い方

```python 
from termcolor import colored

print('test')
print(colored('test', 'red'))

```

- pypiの説明ページ
https://pypi.org/project/termcolor/

- print(help(termcolor)) でヘルプ表示
