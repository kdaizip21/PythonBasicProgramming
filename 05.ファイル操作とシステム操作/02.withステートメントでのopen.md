## OpenしたらCloseする
- `close`しないと、余計なリソース（メモリ）を確保することになり、無駄
- 必ず`close`すること

```python
f = open('test.txt', 'w')

f.write('Test\n')

f.close()
```


## withはcloseしなくていい
- `with`ステートメントを利用すると、`close`が必要ない
- ***`Close`忘れ防止のため、`With`利用することが推奨される***


```python
with open('test.txt', 'w') as f:
    f.write('Test\n')
```

1. `f=`を`with + as f`に書き換えるだけ
2. 処理はインデント
3. `close`不要
