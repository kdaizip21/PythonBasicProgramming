### `tell()`で現在位置を表示  
```python
with open('test.txt', 'r') as f:
    print(f'現在のカーソル位置 : {f.tell()}')
    # 一文字読み込み
    print(f.read(1))
```
```sh
現在のカーソル位置 : 0
A
```

### `seek()`で文字数移動
```python
with open('test.txt', 'r') as f:
    print(f'現在のカーソル位置 : {f.tell()}')
    # 一文字読み込み
    print(f.read(1))

    f.seek(5)
    print(f'現在のカーソル位置 : {f.tell()}')
    print(f.read(1))
```
```sh
現在のカーソル位置 : 0
A
現在のカーソル位置 : 5
B
```
※ 改行コードも1文字扱い
