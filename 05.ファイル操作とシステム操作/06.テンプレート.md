## 文字列は読み取り専用で、下記の文字列は変わらないように扱いたい
```python
s = """\

Hi $name.

$contents

Have a good day
"""
```
- string.Templateを使う
   
- 特定の文字列の`$`に代入する

## `string.Template()`で文字列を読み込む
```python
import string

s = """\

Hi $name.

$contents

Have a good day
"""

t = string.Template(s)
contents = t.substitute(name='Mike', contents='How are you?')
print(contents)
```
```sh
Hi Mike.

How are you?

Have a good day
```

- `string.Template()`
- `substitude()`
- の組み合わせる

## テンプレートファイルは外から呼び出す場合が多い

- template.txt
    ```
    Hi $name.

    $contents

    Have a good day
    ```

- study.py
    ```python
    import string

    with open('template.txt') as f:
        t = string.Template(f.read())

    contents = t.substitute(name='Mike', contents='How are you?')
    print(contents)
    ```
