### 辞書のkeyとvalueをまとめて取る

- 辞書がある
```python
d = {'x': 100, 'y': 200}
```

- 普通に取り出す
```python 
for v in d:
    print(v)
```
```sh
x
y
```
Keyだけ取れた

### 辞書のitems()メソッド
- items()を使って、key、value両方取る
```python 
for k, v in d.items():
    print(k, ':', v)
```
```sh
x : 100
y : 200
```
両方取れる


### items()の中身
```python
print(d.items())

dict_items([('x', 100), ('y', 200)])
```

`リスト`の中に`タプル`が入ってる状態。


1. 最初のforループでリストの最初`('x', 100)`がタプルで取り出される

2. タプルのアンパッキングで`k`, `v`それぞれに`x` , `100`が割り当たる
